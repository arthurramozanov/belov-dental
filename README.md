# Стоматология им. Белова

Лендинг авторской премиум-стоматологии. Демо-проект для портфолио.

## Стек

Next.js 14 (App Router), TypeScript, Tailwind CSS, shadcn/ui, Framer Motion, React Hook Form + Zod.

## Запуск

```bash
pnpm install
pnpm dev
```

Открыть http://localhost:3005

## Production-билд

```bash
pnpm build
pnpm start
```
